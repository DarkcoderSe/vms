## Fruit comission management system

#### How to start project?

Please, follow these steps to properly install project.

* Clone project using ``` git clone https://github.com/DarkcoderSe/fruit-comission-management-system.git ```
* Download Packages using ``` Composer install ```
* Migrate the Tables ``` php artisan migrate ```
* Install Passport (Only For Android/iOS APIs) ``` php artisan passport:install ```
